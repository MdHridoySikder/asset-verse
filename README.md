# Project Name

AssetVerse — Corporate Asset Management System

## Project Description

AssetVerse is a web-based platform designed to manage corporate assets efficiently. It allows admins to track, assign, and manage company assets, while employees can view and request assets. The system ensures transparency, accountability, and streamlined asset allocation within organizations.

Users can explore:

- **Assets:** View available assets and their types (Returnable / Non-returnable).
- **Requests:** Submit asset requests and track approval status.
- **Teams & Employees:** Admins can manage teams and employees.
- **Profile & Settings:** Update personal information and manage account settings.

### Features

- Admin dashboard for managing assets, requests, and employees.
- Employee dashboard for requesting and tracking assets.
- Asset classification into Returnable and Non-returnable types.
- Real-time request approvals and tracking.
- Data visualization using charts (e.g., asset type distribution).
- Responsive design for desktop and mobile devices.
- Interactive UI with React Router and Tailwind CSS.

### Technologies Used

- **React** – Frontend framework
- **React Router** – Page routing
- **Tailwind CSS** – Styling and responsive design
- **DaisyUI** – Prebuilt Tailwind components
- **React Query** – Data fetching and caching
- **Axios** – HTTP client for API calls
- **Firebase** – Authentication and user management
- **Firebase Admin SDK** – Admin server tasks
- **MongoDB** – Database for assets, requests, and employees
- **Express.js** – Backend API server
- **Node.js** – Server runtime
- **Vercel** – Server hosting
- **React Toastify** – Notifications
- **Recharts** – Charts and data visualization
- **React Icons** – UI icons
- **React Hook Form** – Form handling
- **SweetAlert2** – Interactive alerts
- **Swiper / React Responsive Carousel** – Image sliders and carousels
- **Leaflet / React-Leaflet** – Map integration
- **Git & GitHub** – Version control
- **VS Code** – Code editor
- **Google Fonts** – Custom web fonts
